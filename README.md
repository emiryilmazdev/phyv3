# Şimal Ongun - Fizyoterapist Website

Modern Next.js 16 ile geliştirilmiş fizyoterapi kliniği web sitesi.

## Teknolojiler

- **Next.js 16** (App Router)
- **TypeScript**
- **React 19**
- **Bootstrap 5.3**
- **GSAP** (Animasyonlar)
- **Framer Motion**
- **Lenis** (Smooth Scrolling)

## Kurulum

```bash
npm install
npm run dev
```

## Deployment

Vercel'de otomatik deploy edilir.

## Özellikler

- Responsive tasarım
- Modern animasyonlar
- SEO optimizasyonu
- Performance optimizasyonu
- TypeScript desteği
